# INI-005: セルフ貸出・ICタグ対応イニシアチブ（Phase 4）

最終更新: 2024-04-22

---

## 概要

本イニシアチブは、**完全デジタル化と無人運用の基盤構築** を目的とする。
セルフ貸出端末の導入、ICタグによる蔵書管理、学校図書館との連携を実現し、
利用者の利便性向上と職員の業務効率化を推進する。

---

## ビジネス目標

| 項目 | 内容 |
|------|------|
| **目的** | 利用者自身による貸出処理の実現、蔵書管理の自動化 |
| **成功指標** | セルフ貸出利用率50%、蔵書棚卸し時間80%削減 |
| **対象者** | 図書館利用者、図書館職員 |
| **期待効果** | 待ち時間削減、職員の付加価値業務へのシフト |

---

## スコープ

### In Scope（対象範囲）

| カテゴリ | 内容 | 優先度 |
|----------|------|--------|
| **セルフ貸出端末** | 利用者自身による貸出・返却処理 | Must |
| **ICタグ管理** | 蔵書へのICタグ貼付、一括読取 | Must |
| **自動返却ポスト** | 閉館時も返却可能な返却ボックス | Should |
| **学校図書館連携** | 市内小中学校との蔵書共有検索 | Should |

### Out of Scope（対象外）

| カテゴリ | 理由 |
|----------|------|
| 電子書籍貸出 | 将来検討 |
| 自動書架ロボット | 費用対効果検討中 |
| 他自治体連携 | 将来検討 |

---

## 前提条件

- INI-002〜INI-004が完了し、安定稼働していること
- セルフ貸出端末の設備投資予算が確保されていること
- ICタグ（RFIDタグ）の導入予算が確保されていること

---

## 主要機能

### 1. セルフ貸出端末

| 機能 | 説明 |
|------|------|
| 利用者カード読取 | バーコード/QRコードでの本人確認 |
| 図書読取 | ICタグ/バーコードでの図書識別 |
| 一括貸出 | 複数冊同時読取・貸出 |
| 音声ガイド | 操作案内の音声出力 |
| 多言語対応 | 日本語、英語、中国語、韓国語 |

### 2. ICタグ管理

| 機能 | 説明 |
|------|------|
| 蔵書タグ貼付 | 全蔵書へのICタグ導入 |
| ゲート検知 | 未貸出図書の持ち出し検知 |
| 一括棚卸し | ハンディリーダーによる蔵書点検 |
| 配架確認 | 正しい棚に配架されているか確認 |

### 3. 学校図書館連携

| 機能 | 説明 |
|------|------|
| 蔵書共有検索 | 市内小中学校の蔵書を検索可能 |
| 巡回配送 | 学校間・中央図書館間の資料配送 |
| 利用者ID連携 | 学校図書館カードとの紐付け |
