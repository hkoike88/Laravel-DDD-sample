# 青空市立図書館 業務システム プロトタイプ

INI-001（図書館業務システム刷新）のUIプロトタイプです。

## 概要

このプロトタイプは、青空市立中央図書館の業務システムのUI/UXを検証するためのものです。
React + Vite + TypeScript で構築されており、バックエンドなしで動作します（モックデータ使用）。

## 対象機能（MVP Phase 1）

### EP-01: 蔵書管理
- **UC-001-001**: 蔵書検索 - キーワード・カテゴリ・ステータスでの検索
- **UC-001-002**: 蔵書登録 - ISBN検索、手動入力、バーコードスキャン対応

### EP-02: 貸出・返却
- **UC-001-003**: 貸出処理 - 利用者確認、蔵書選択、貸出実行
- **UC-001-004**: 返却処理 - 延滞チェック、予約確認、取り置き処理

### EP-03: 予約管理
- **UC-001-005**: 予約登録 - 利用者への予約受付
- **UC-001-006**: 予約管理 - 取り置き状況確認、キャンセル処理

### EP-04: 利用者管理
- **UC-001-007**: 利用者登録 - 新規利用者の登録
- **UC-001-008**: 利用者検索 - 利用者情報の検索・詳細確認

## 業務ルール

| ルールID | 内容 |
|----------|------|
| BR-001 | 貸出上限: 1人5冊まで |
| BR-002 | 貸出期間: 一般図書14日間、新刊・雑誌・AV7日間 |
| BR-003 | 延滞中は新規貸出停止 |
| BR-004 | 予約上限: 1人3冊まで |
| BR-005 | 取り置き期間: 連絡後7日間 |
| BR-006 | 利用者登録資格: 市内在住・在勤・在学者 |
| BR-007 | カード有効期限: 登録から1年間 |

## 技術スタック

- **フレームワーク**: React 19.x
- **ビルドツール**: Vite 7.x
- **言語**: TypeScript 5.x
- **ルーティング**: React Router DOM 7.x
- **スタイル**: CSS（ワイヤーフレーム風）

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 制限事項

- **データ永続化なし**: 操作結果は保存されません（画面遷移・リロードでリセット）
- **認証なし**: ログイン機能は実装されていません
- **バックエンド連携なし**: すべてモックデータで動作

## 検証ポイント

このプロトタイプで確認すべき点:

1. **画面遷移フロー**: 業務の流れに沿った画面遷移が適切か
2. **情報設計**: 必要な情報が適切に表示されているか
3. **操作性**: 職員が直感的に操作できるか
4. **業務ルール表示**: 制約条件が分かりやすく表示されているか
5. **エラーハンドリング**: エラー状態の表示が適切か
