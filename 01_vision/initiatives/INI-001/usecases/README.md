## アクター

| アクター | 説明 | 関連 Use Case |
|----------|------|---------------|
| 職員 | 図書館職員（一般権限） | UC-001-001, UC-001-002, UC-001-006, UC-001-008〜011 |
| 管理者 | システム管理者（管理権限） | UC-001-003, UC-001-004, UC-001-005 |
| 開発者 | セキュリティ設定を実装する | UC-001-007 |

※ 一般利用者向けログイン・マイページ機能は Phase 3 以降で検討
※ 蔵書検索（公開機能）は INI-002 で対応
※ 利用者管理機能は職員権限で実行可能（管理者権限は不要）

## 前提条件

- INI-000（開発環境構築）が完了していること
- EPIC-005（職員認証機能）が実装済みであること
- EPIC-006（職員ログアウト機能）が実装済みであること
- EPIC-007（セキュリティ対策準備）が適用済みであること

## 備考

- 各 Use Case は Epic・Story に展開され、`02_backlog/` で管理される
- 本イニシアチブは INI-002（図書館業務デジタル化 MVP）の前提条件となる
- 職員ログイン機能は EPIC-005 で実装済みの認証 API を利用する
- 蔵書検索（公開機能）は INI-002 で対応
- **一般利用者向けログイン・マイページ機能は Phase 3 以降で検討**
