## UI 要素一覧

| 要素ID | 種類 | 要素名 | 必須 | 編集可 | 説明 |
|--------|------|--------|------|--------|------|
| patron-id | text | 利用者番号 | - | ✗ | 表示のみ |
| name | input[text] | 氏名 | ○ | ○ | 姓名をスペース区切り |
| name-kana | input[text] | ふりがな | ○ | ○ | ひらがなで入力 |
| birth-date | text | 生年月日 | - | ✗ | 表示のみ |
| patron-type | radio | 利用者区分 | ○ | ○ | 一般/学生/児童 |
| postal-code | input[text] | 郵便番号 | - | ○ | ハイフンあり/なし両対応 |
| address | input[text] | 住所 | ○ | ○ | 郵便番号から自動入力可 |
| phone | input[tel] | 電話番号 | ○ | ○ | 連絡先として使用 |
| notes | textarea | 備考 | - | ○ | 自由記述 |
| btn-cancel | button | キャンセル | - | - | 詳細画面に戻る |
| btn-submit | button | 保存する | - | - | 変更を保存 |

## 入力項目詳細

| 項目 | 必須 | 形式 | バリデーション | 編集可 |
|------|:----:|------|---------------|:------:|
| 利用者番号 | - | - | - | ✗ |
| 氏名 | ○ | 文字列 | 50文字以内 | ○ |
| ふりがな | ○ | ひらがな | 50文字以内 | ○ |
| 生年月日 | - | - | - | ✗ |
| 利用者区分 | ○ | 選択 | - | ○ |
| 郵便番号 | - | 数字 | 7桁 | ○ |
| 住所 | ○ | 文字列 | 200文字以内 | ○ |
| 電話番号 | ○ | 電話番号 | - | ○ |
| 備考 | - | 文字列 | 500文字以内 | ○ |

---

## 変更不可項目

- **利用者番号**: 一度採番されたら変更不可
- **生年月日**: 本人確認の基準となるため変更不可（誤登録の場合は管理者に連絡）

---

## 画面遷移

| アクション | 遷移先 | 備考 |
|-----------|--------|------|
| キャンセル | `/staff/patrons/{id}` | 確認ダイアログなし |
| 保存成功 | `/staff/patrons/{id}` | 成功メッセージ表示 |
| 保存失敗 | 現在の画面 | エラーメッセージ表示 |

---

## 変更履歴

- 変更内容は自動的に履歴として記録される
- 履歴は管理者のみ閲覧可能（Phase 2で実装）

---

## 住所変更時の注意

住所が対象地域外（市外）に変更された場合：

