## UI 要素一覧

| 要素ID | 種類 | 要素名 | 必須 | 説明 |
|--------|------|--------|------|------|
| name | input[text] | 氏名 | ○ | 姓名をスペース区切り |
| name-kana | input[text] | ふりがな | ○ | ひらがなで入力 |
| birth-date | select×3 | 生年月日 | ○ | 年月日を選択 |
| patron-type | radio | 利用者区分 | ○ | 一般/学生/児童 |
| postal-code | input[text] | 郵便番号 | - | ハイフンあり/なし両対応 |
| address | input[text] | 住所 | ○ | 郵便番号から自動入力可 |
| phone | input[tel] | 電話番号 | ○ | 連絡先として使用 |
| notes | textarea | 備考 | - | 通勤・通学先など |
| btn-cancel | button | キャンセル | - | 一覧画面に戻る |
| btn-submit | button | 登録する | - | 登録を実行 |

## 入力項目詳細

| 項目 | 必須 | 形式 | バリデーション | エラーメッセージ |
|------|:----:|------|---------------|----------------|
| 氏名 | ○ | 文字列 | 50文字以内 | 氏名を入力してください |
| ふりがな | ○ | ひらがな | 50文字以内 | ふりがなをひらがなで入力してください |
| 生年月日 | ○ | 日付 | 未来日不可 | 正しい生年月日を入力してください |
| 利用者区分 | ○ | 選択 | - | 利用者区分を選択してください |
| 郵便番号 | - | 数字 | 7桁 | 正しい郵便番号を入力してください |
| 住所 | ○ | 文字列 | 200文字以内 | 住所を入力してください |
| 電話番号 | ○ | 電話番号 | - | 正しい電話番号を入力してください |
| 備考 | - | 文字列 | 500文字以内 | - |

## 画面遷移

| アクション | 遷移先 | 備考 |
|-----------|--------|------|
| キャンセル | `/staff/patrons` | 確認ダイアログなし |
| 登録成功→閉じる | `/staff/patrons/{id}` | 詳細画面へ |
| 登録成功→カード印刷 | 印刷ダイアログ | 印刷後に詳細画面へ |

---

## ビジネスルール

- 登録対象は市内在住・在勤・在学者
- 有効期限は登録日から1年間
- 利用者番号は「P」+8桁の連番で自動採番
- 児童（小学生以下）は保護者情報を備考に記入推奨
