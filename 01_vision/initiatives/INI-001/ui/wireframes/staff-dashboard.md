## UI 要素一覧

### ヘッダー

| 要素ID | 種類 | 要素名 | 説明 |
|--------|------|--------|------|
| logo | link | システムロゴ | クリックでダッシュボードへ |
| nav-patrons | link | 利用者管理 | 利用者管理画面へ遷移 |
| nav-books | link | 蔵書管理 | 蔵書管理画面へ遷移（INI-002） |
| nav-loans | link | 貸出・返却 | 貸出・返却画面へ遷移（INI-002） |
| user-menu | dropdown | 職員名 | ログアウト等のメニュー |

### メインコンテンツ

| 要素ID | 種類 | 要素名 | 説明 |
|--------|------|--------|------|
| welcome | text | ウェルカムメッセージ | ログイン職員名と日付を表示 |
| card-patrons | card | 利用者管理カード | 利用者管理機能へのショートカット |
| card-books | card | 蔵書管理カード | 蔵書管理機能へのショートカット（INI-002） |
| card-lending | card | 貸出処理カード | 貸出処理機能へのショートカット（INI-002） |
| card-return | card | 返却処理カード | 返却処理機能へのショートカット（INI-002） |
| card-reservations | card | 予約管理カード | 予約管理機能へのショートカット（INI-002） |
| card-staff | card | 職員管理カード | 職員アカウント管理へ（管理者のみ表示） |
| notices | section | お知らせ | システムからのお知らせ一覧 |
| daily-stats | section | 本日の状況 | 当日の貸出・返却・延滞件数 |

## 機能カード詳細

| カード | 遷移先 | 実装フェーズ | 表示条件 |
|--------|--------|-------------|----------|
| 利用者管理 | `/staff/patrons` | INI-001 | 全職員 |
| 蔵書管理 | `/staff/books` | INI-002 | 全職員 |
| 貸出処理 | `/staff/loans/new` | INI-002 | 全職員 |
| 返却処理 | `/staff/loans/return` | INI-002 | 全職員 |
| 予約管理 | `/staff/reservations` | INI-002 | 全職員 |
| 職員管理 | `/staff/accounts` | INI-001 | 管理者のみ |

## 画面遷移

| アクション | 遷移先 | 備考 |
|-----------|--------|------|
| ロゴクリック | `/staff/dashboard`（現在の画面） | - |
| 利用者管理クリック | `/staff/patrons` | INI-001 で実装 |
| 蔵書管理クリック | `/staff/books` | INI-002 で実装 |
| 貸出・返却クリック | `/staff/loans` | INI-002 で実装 |
| 職員管理クリック | `/staff/accounts` | INI-001 で実装、管理者のみ |
| ログアウト | `/staff/login` | セッション終了後 |

## データ表示

### 本日の状況

| 項目 | API エンドポイント | 説明 |
|------|-------------------|------|
| 貸出件数 | `GET /api/stats/today/loans` | 当日の貸出件数 |
| 返却件数 | `GET /api/stats/today/returns` | 当日の返却件数 |
| 延滞件数 | `GET /api/stats/overdue` | 現在の延滞件数 |

> **Note**: 統計 API は INI-002 で実装予定。INI-001 ではプレースホルダー（--）を表示。

## アクセシビリティ

- ナビゲーションは `nav` ランドマークでマークアップ
- メインコンテンツは `main` ランドマークでマークアップ
- 機能カードはキーボードでフォーカス・選択可能
- 色だけでなくアイコンで機能を識別可能
