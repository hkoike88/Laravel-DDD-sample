## UI 要素一覧

| 要素ID | 種類 | 要素名 | 必須 | 説明 |
|--------|------|--------|------|------|
| name | input[text] | 氏名 | ○ | 職員の氏名 |
| email | input[email] | メールアドレス | - | 表示のみ（変更不可） |
| role | select | 権限 | ○ | 一般職員/管理者 |
| status | text | ステータス | - | 有効/無効（表示のみ） |
| btn-reset-password | button | パスワードリセット | - | リセットメール送信 |
| btn-deactivate | button | アカウント無効化 | - | 無効化確認ダイアログ表示 |
| btn-cancel | button | キャンセル | - | 一覧画面に戻る |
| btn-submit | button | 保存する | - | 変更を保存 |

---

## 入力項目

| 項目 | 必須 | 形式 | バリデーション | 備考 |
|------|:----:|------|---------------|------|
| 氏名 | ○ | 文字列 | 50文字以内 | 変更可能 |
| メールアドレス | - | - | - | 変更不可（表示のみ） |
| 権限 | ○ | 選択 | - | 変更可能 |

## 画面遷移

| アクション | 遷移先 | 備考 |
|-----------|--------|------|
| キャンセル | `/staff/accounts` | 確認ダイアログなし |
| 保存成功 | `/staff/accounts` | 成功メッセージ表示 |
| パスワードリセット成功 | 現在の画面 | 成功メッセージ表示 |
| 無効化成功 | `/staff/accounts` | 成功メッセージ表示 |

---

## ビジネスルール

- 自分自身のアカウントは無効化できない
- 最後の管理者アカウントは無効化できない
- メールアドレスは変更不可（セキュリティ上の理由）
