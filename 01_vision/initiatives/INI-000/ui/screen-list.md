# INI-000: 画面一覧

最終更新: 2025-12-24

---

## 概要

本ドキュメントは、INI-000（開発環境構築・認証基盤）で実装する画面の一覧を定義する。

---

## 画面一覧

| 画面ID | 画面名 | URL パス | 認証 | 関連ユースケース | 説明 |
|--------|--------|----------|------|-----------------|------|
| SCR-000-001 | ログイン画面 | `/login` | 不要 | UC-000-005 | 職員がシステムにログインする |
| SCR-000-002 | ダッシュボード | `/` | 必須 | - | ログイン後のトップ画面、各機能へのナビゲーション |

---

## 画面詳細

### SCR-000-001: ログイン画面

| 項目 | 内容 |
|------|------|
| 画面ID | SCR-000-001 |
| 画面名 | ログイン画面 |
| URL パス | `/login` |
| 認証要否 | 不要（未ログイン状態でアクセス可能） |
| 関連ユースケース | [UC-000-005 職員ログイン](../usecases/UC-000-005_職員ログイン.md) |

**機能概要:**
- 職員ID（メールアドレス）とパスワードによる認証
- ログイン成功時はダッシュボードへリダイレクト
- ログイン失敗時はエラーメッセージを表示

**ワイヤーフレーム:** [login.md](./wireframes/login.md)

---

### SCR-000-002: ダッシュボード

| 項目 | 内容 |
|------|------|
| 画面ID | SCR-000-002 |
| 画面名 | ダッシュボード |
| URL パス | `/` |
| 認証要否 | 必須（未ログイン時はログイン画面へリダイレクト） |
| 関連ユースケース | - |

**機能概要:**
- ログイン後のトップ画面
- 各機能（蔵書管理、貸出・返却、利用者管理など）へのナビゲーション
- ログイン中の職員情報を表示
- ログアウト機能

**ワイヤーフレーム:** [dashboard.md](./wireframes/dashboard.md)

---

## 共通レイアウト

### ヘッダー（認証後）

| 要素 | 説明 |
|------|------|
| システムロゴ | クリックでダッシュボードへ遷移 |
| ナビゲーション | 蔵書管理、貸出・返却、利用者管理へのリンク |
| 職員名表示 | ログイン中の職員名を表示 |
| ログアウトボタン | クリックでログアウト処理を実行 |

### フッター

| 要素 | 説明 |
|------|------|
| コピーライト | システム名・年度表示 |
