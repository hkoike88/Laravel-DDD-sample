## UI 要素一覧

### ヘッダー

| 要素ID | 種類 | 要素名 | 説明 |
|--------|------|--------|------|
| logo | link | システムロゴ | クリックでダッシュボードへ |
| nav-books | link | 蔵書管理 | 蔵書管理画面へ遷移 |
| nav-loans | link | 貸出・返却 | 貸出・返却画面へ遷移 |
| nav-users | link | 利用者管理 | 利用者管理画面へ遷移 |
| user-menu | dropdown | 職員名 | ログアウト等のメニュー |

| 要素ID | 種類 | 要素名 | 説明 |
|--------|------|--------|------|

| card-books | card | 蔵書管理カード | 蔵書管理機能へのショートカット |
| card-lending | card | 貸出処理カード | 貸出処理機能へのショートカット |
| card-return | card | 返却処理カード | 返却処理機能へのショートカット |

## 画面遷移

| アクション | 遷移先 | 備考 |
|-----------|--------|------|
| ロゴクリック | `/`（現在の画面） | - |
| 蔵書管理クリック | `/books` | INI-001 で実装 |
| 貸出・返却クリック | `/loans` | INI-001 で実装 |
| 利用者管理クリック | `/users` | INI-001 で実装 |
| ログアウト | `/login` | セッション終了後 |

## データ表示

### 本日の状況

| 項目 | API エンドポイント | 説明 |
|------|-------------------|------|
| 貸出件数 | `GET /api/stats/today/loans` | 当日の貸出件数 |
| 返却件数 | `GET /api/stats/today/returns` | 当日の返却件数 |
| 延滞件数 | `GET /api/stats/overdue` | 現在の延滞件数 |

> **Note**: 統計 API は Phase 2 で実装予定。INI-000 ではモックデータを表示。

## アクセシビリティ

- ナビゲーションは `nav` ランドマークでマークアップ
- メインコンテンツは `main` ランドマークでマークアップ
- 機能カードはキーボードでフォーカス・選択可能
- 色だけでなくアイコンで機能を識別可能
