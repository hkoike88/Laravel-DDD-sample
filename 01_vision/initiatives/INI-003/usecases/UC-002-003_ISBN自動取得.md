## 概要

職員が蔵書登録時にISBNを入力すると、外部API（OpenBD等）から書籍情報を自動取得し、入力を省力化する。

---

## 事前条件

- 職員がログインしていること
- 外部APIが利用可能であること
- 登録する図書にISBNが付与されていること

---

## 事後条件

- ISBNから取得した書籍情報がフォームに自動入力される
- 職員が内容を確認・修正して登録できる

---

## 基本フロー

1. 職員が蔵書登録画面を開く
2. 職員がISBNを入力する
3. システムが外部APIに問い合わせる
4. システムが取得した書籍情報（タイトル、著者、出版社、発行年等）をフォームに自動入力する
5. 職員が内容を確認・修正する
6. 職員が登録ボタンをクリックする

---

## 代替フロー

### 4a. API取得失敗の場合

1. システムが「書籍情報を取得できませんでした」と表示する
2. 職員が手動で情報を入力する
3. 基本フロー5に進む

### 4b. ISBNが見つからない場合

1. システムが「該当する書籍が見つかりませんでした」と表示する
2. 職員が手動で情報を入力する
3. 基本フロー5に進む

---

## 取得情報

| 項目 | 取得元 |
|------|--------|
| タイトル | OpenBD API |
| 著者名 | OpenBD API |
| 出版社 | OpenBD API |
| 発行年 | OpenBD API |
| 表紙画像URL | OpenBD API |
| ページ数 | OpenBD API |

## ビジネスルール

| ルールID | ルール内容 |
|----------|-----------|
| BR-UC203-01 | APIからの取得情報は編集可能とする |
| BR-UC203-02 | API取得失敗時は手動入力にフォールバックする |
| BR-UC203-03 | APIレスポンスは5秒以内にタイムアウトする |
