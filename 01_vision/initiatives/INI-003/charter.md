# INI-003: 機能拡張イニシアチブ（Phase 2）

最終更新: 2024-04-22

---

## 概要

本イニシアチブは、MVP（INI-002）完了後の **運用効率向上とデータ活用の開始** を目的とする。
統計レポート機能、遅延通知メール、貸出上限チェック強化、ISBN API連携などの機能拡張を行う。

---

## ビジネス目標

| 項目 | 内容 |
|------|------|
| **目的** | 運用効率のさらなる向上、データに基づくサービス改善の基盤構築 |
| **成功指標** | 月次統計レポート自動生成、遅延通知による返却率向上 |
| **対象者** | 図書館職員、利用者 |
| **期待効果** | レポート作成時間削減、督促業務の自動化 |

---

## スコープ

### In Scope（対象範囲）

| カテゴリ | 内容 | 優先度 | 対応課題 |
|----------|------|--------|----------|
| **統計レポート** | 日別貸出冊数、人気本ランキング、月次レポート | Must | ISS-012 |
| **遅延通知メール** | 返却期限超過時の自動メール通知 | Should | ISS-013 |
| **貸出上限チェック強化** | 利用者カテゴリ別の貸出制限 | Should | - |
| **ISBN API連携** | 書籍情報の自動取得（OpenBD等） | Could | - |

### Out of Scope（対象外）

| カテゴリ | 理由 |
|----------|------|
| スマートフォン対応 | Phase 3（INI-004）で対応 |
| マイページ機能 | Phase 3（INI-004）で対応 |
| セルフ貸出 | Phase 4（INI-005）で対応 |

---

## 前提条件

- INI-002（MVP）が完了し、安定稼働していること
- 貸出データが蓄積されていること（統計レポート用）
- メールサーバーが利用可能であること

---

## 主要機能

### 1. 統計レポート機能

| 機能 | 説明 |
|------|------|
| 日別貸出統計 | 日ごとの貸出冊数、来館者数を自動集計 |
| 人気本ランキング | 月間/年間の貸出回数ランキング |
| 月次レポート | 貸出統計、利用者統計の月次サマリー |
| エクスポート | CSV/PDF形式での出力 |

### 2. 遅延通知機能

| 機能 | 説明 |
|------|------|
| 返却期限リマインダー | 返却期限3日前にメール通知 |
| 延滞通知 | 返却期限超過時に自動メール通知 |
| 督促状生成 | 長期延滞者への督促状テンプレート |

### 3. ISBN API連携

| 機能 | 説明 |
|------|------|
| 書籍情報自動取得 | ISBNから書籍情報（タイトル、著者等）を自動取得 |
| 表紙画像取得 | 書籍の表紙画像を取得・表示 |
