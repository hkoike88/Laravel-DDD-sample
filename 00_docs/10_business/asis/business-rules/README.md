## 目的

このドキュメントは、現行の図書館業務で適用されている **業務ルール** を整理します。

- 現場の「暗黙のルール」を可視化
- ステークホルダーとの認識合わせ
- TO-BE設計時の参考情報

**注意**: これは現行ルールの記録であり、新システムの仕様ではありません。
システム実装時のルールは、Epic着手時に別途定義します。

## ルール詳細

### BR-001: 貸出上限

| 項目 | 内容 |
|------|------|
| ルール | 1人5冊まで同時に借りられる |
| 確認方法 | 貸出管理Excelで手作業でカウント |
| 例外 | 団体貸出は別枠（50冊まで） |
| 情報源 | 山田さん（ベテラン司書）へのヒアリング |

#### 運用実態

| 項目 | 内容 |
|------|------|
| 確認タイミング | 貸出処理時にExcelをフィルタしてカウント |
| 所要時間 | 1〜3分（利用者によって異なる） |
| 課題 | Excelが重い、フィルタの手間、カウントミスのリスク |
| 現場の声 | 「常連さんは覚えているが、たまに来る人は確認が大変」（山田さん） |

---

### BR-002: 貸出期間

| 項目 | 内容 |
|------|------|
| ルール | 一般図書は14日間、新刊・雑誌・AVは7日間 |
| 延長 | 一般図書のみ1回可（予約なしの場合） |
| 確認方法 | 返却期限スタンプで管理 |
| 情報源 | 図書館利用規則 第5条 |

#### 資料区分別の貸出期間

| 資料区分 | 貸出期間 | 延長 | 判定方法 |
|---------|---------|------|---------|
| 一般図書 | 14日間 | 1回可 | 通常の図書 |
| 新刊図書 | 7日間 | 不可 | 発売後3ヶ月以内 |
| 雑誌（バックナンバー） | 7日間 | 不可 | 最新号以外 |
| CD・DVD | 7日間 | 不可 | 視聴覚資料 |

#### 新刊判定について

| 項目 | 内容 |
|------|------|
| 判定基準 | 発売日から3ヶ月以内 |
| 確認方法 | 奥付の発行日を目視確認、または新刊コーナーに配架されているか |
| 運用実態 | 新刊コーナーに置いている間は7日間貸出として運用 |
| 課題 | 3ヶ月経過の判断が曖昧、担当者によってばらつきあり |
| 情報源 | 山田さんへのヒアリング |

---

### BR-003: 延滞対応

| 項目 | 内容 |
|------|------|
| ルール | 延滞中は新規貸出停止 |
| 督促 | 1週間後に電話、2週間後に督促状 |
| 常習者対応 | 年3回以上で館長面談 |
| 情報源 | 図書館利用規則 第8条 |

#### 運用実態

| 項目 | 内容 |
|------|------|
| 延滞把握 | 貸出時にExcelを確認して初めて気づく |
| 督促実施 | 忙しいと後回しになることがある |
| 常習者把握 | 記憶頼み、年間回数は正確に把握できていない |
| 課題 | 延滞者リストがない、督促漏れが発生する |
| 現場の声 | 「督促の電話は気が重い。システムで自動通知してほしい」（佐藤さん） |

---

### BR-004: 予約上限

| 項目 | 内容 |
|------|------|
| ルール | 1人3冊まで、1タイトル3人まで |
| 確認方法 | ホワイトボードで目視確認 |
| 情報源 | 高橋さん（副館長）へのヒアリング |

#### 運用実態

| 項目 | 内容 |
|------|------|
| 確認方法 | ホワイトボード全体を見て、同じ人の名前をカウント |
| 所要時間 | 1〜2分 |
| 課題 | ホワイトボードが見づらい、同姓同名の区別が困難 |
| 現場の声 | 「予約が多い時期はホワイトボードがいっぱいになる」（佐藤さん） |

---

### BR-005: 取り置き期限

| 項目 | 内容 |
|------|------|
| ルール | 連絡完了から7日間取り置き |
| 連絡方法 | 電話（つながらなければ3回まで試行） |
| 期限切れ対応 | 次の予約者へ、または書架へ |
| 情報源 | 佐藤さん（若手司書）へのヒアリング |

#### 運用実態

| 項目 | 内容 |
|------|------|
| 連絡タイミング | 返却処理後、手が空いたときに電話 |
| 期限管理 | 取り置き棚の本に付箋で日付を記入、毎朝目視確認 |
| 電話不通時 | 留守電に残す、翌日再連絡 |
| 課題 | 連絡先メモ漏れ、期限切れの見落とし、電話がつながりにくい |
| 現場の声 | 「日中は留守の方が多い。メールで連絡できれば楽なのに」（中村さん） |

---

### BR-006: 利用者登録資格

| 項目 | 内容 |
|------|------|
| ルール | 市内在住・在勤・在学者が対象 |
| 必要書類 | 本人確認書類（在勤・在学は追加書類） |
| 広域利用 | 近隣3市町村との協定あり |
| 情報源 | 図書館設置条例 第3条 |

#### 運用実態

| 項目 | 内容 |
|------|------|
| 確認方法 | 本人確認書類の住所を目視確認 |
| 在勤・在学確認 | 社員証や学生証を目視確認 |
| 広域利用 | 協定市町村のリストを見て確認（リストは事務室に掲示） |
| 課題 | 書類の種類が多く確認に時間がかかる、広域協定の対象か迷うことがある |
| 現場の声 | 「最近マイナンバーカードが増えて確認しやすくなった」（山田さん） |

---

### BR-007: カード有効期限

| 項目 | 内容 |
|------|------|
| ルール | 登録日から1年間有効 |
| 更新 | 本人確認で更新可（期限切れ1ヶ月以内） |
| 期限管理 | 手作業（把握できていないことも多い） |
| 情報源 | 山田さんへのヒアリング |

#### 運用実態

| 項目 | 内容 |
|------|------|
| 確認タイミング | 貸出時にカードの有効期限スタンプを目視確認 |
| 期限切れ対応 | その場で更新手続き（本人確認書類があれば） |
| 事前通知 | なし（来館時に気づく） |
| 課題 | 期限切れに気づかず来館する利用者が多い、更新手続きで待ち時間が発生 |
| 現場の声 | 「期限切れで借りられないと残念そうにされる。事前にお知らせできれば」（中村さん） |

---

### BR-008: 禁帯出資料

| 項目 | 内容 |
|------|------|
| ルール | 参考図書・郷土資料・雑誌最新号・新聞・貴重書は館内のみ |
| 確認方法 | 背ラベルの色で判別（赤ラベル＝禁帯出） |
| 例外 | 館長許可で持ち出し可（展示・調査目的） |
| 情報源 | 現地視察で確認 |

#### 運用実態

| 項目 | 内容 |
|------|------|
| 判別方法 | 背ラベルの色（赤＝禁帯出）、配架場所（参考図書コーナー等） |
| 貸出時確認 | 貸出処理時に目視で確認 |
| 雑誌最新号 | 新しい号が届いたら前の号を貸出可能に変更（ラベル貼り替え） |
| 課題 | ラベルの貼り忘れ、雑誌の最新号更新作業が手間 |
| 現場の声 | 「禁帯出なのに持って来られることがたまにある。気づかず処理しそうになる」（佐藤さん） |

---

## 暗黙のルール・現場の慣行

ヒアリングや現地視察で把握した、明文化されていないルール。

| 内容 | 情報源 | 備考 |
|------|-------|------|
| 常連利用者には口頭で延長を認めることがある | 山田さん | 正式ルールではない |
| 返却ポストの本は翌朝一括処理 | 佐藤さん | 朝の業務負荷の原因 |
| 予約連絡は午前中に行う慣行 | 中村さん | シフトの関係 |
| 破損本は弁償ではなく注意で済ませることが多い | 山田さん | 利用者との関係維持 |
