# ST-005: 環境変数設定（.env.example）の作成

最終更新: 2025-12-23

---

## ストーリー

**開発者として**、環境変数のテンプレートファイルを使って簡単に環境設定を行いたい。
**なぜなら**、各開発者が自分の環境に合わせてポートやデータベース設定をカスタマイズできるようにしたいからだ。

## 技術仕様

### 環境変数一覧

```bash
# .env.example

# ==============================================
# Docker Compose 設定
# ==============================================

# ポート設定（ホスト側のポート番号）
FRONTEND_PORT=5173
BACKEND_PORT=8000
DB_PORT=3306
PHPMYADMIN_PORT=8080
NGINX_PORT=80

# ==============================================
# データベース設定
# ==============================================

DB_DATABASE=library
DB_USERNAME=library
DB_PASSWORD=secret
DB_ROOT_PASSWORD=secret

# ==============================================
# アプリケーション設定
# ==============================================

# Laravel
APP_ENV=local
APP_DEBUG=true
APP_KEY=

# フロントエンド
VITE_API_BASE_URL=http://localhost:8000/api
```

### .gitignore 追記

```gitignore
# 環境設定ファイル
.env
.env.local
.env.*.local
```

## 使用方法

```bash
# 1. テンプレートをコピー
cp .env.example .env

# 2. 必要に応じて値を編集
vim .env

# 3. Docker Compose を起動
docker compose up -d
```
