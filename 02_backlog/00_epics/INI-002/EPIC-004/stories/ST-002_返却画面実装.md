# ST-002: 返却画面の実装

最終更新: 2025-12-24

---

## ストーリー

**図書館職員として**、返却処理を行う画面を使いたい。
**なぜなら**、利用者からの図書返却を効率的に処理したいからだ。

## 受け入れ条件

1. [ ] バーコード/ISBN で図書を検索できること
2. [ ] 貸出情報（貸出者、貸出日、返却期限）が表示されること
3. [ ] 延滞している場合、警告が表示されること
4. [ ] 予約がある場合、通知が表示されること
5. [ ] 返却完了メッセージが表示されること

|              [返却実行]                   |
+------------------------------------------+
