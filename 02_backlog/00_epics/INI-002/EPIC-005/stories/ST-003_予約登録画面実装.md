# ST-003: 予約登録画面の実装

最終更新: 2025-12-24

---

## ストーリー

**図書館職員として**、予約を登録する画面を使いたい。
**なぜなら**、利用者からの予約依頼を効率的に処理したいからだ。

## 受け入れ条件

1. [ ] 利用者を選択できること
2. [ ] 図書を選択できること
3. [ ] 現在の予約状況（何人待ちか）が表示されること
4. [ ] 予約完了メッセージが表示されること
5. [ ] エラー時に警告が表示されること

```
|         予約登録                          |

|                                          |
| 選択中: 山田太郎 (12345678)               |
+------------------------------------------+
| [図書検索]                                |
| ISBN/タイトル: [________] [検索]          |

+------------------------------------------+
| 予約順: 2番目になります                   |
|                                          |
|              [予約登録]                   |
+------------------------------------------+
```
