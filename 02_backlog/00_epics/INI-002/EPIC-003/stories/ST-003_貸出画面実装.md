# ST-003: 貸出画面の実装

最終更新: 2025-12-24

---

## ストーリー

**図書館職員として**、貸出処理を行う画面を使いたい。
**なぜなら**、利用者への図書貸出を効率的に行いたいからだ。

## 受け入れ条件

1. [ ] 利用者を選択できること（カード番号/氏名で検索）
2. [ ] 図書を選択できること（バーコード/ISBN/タイトルで検索）
3. [ ] 現在の貸出状況が表示されること（貸出数/上限）
4. [ ] 返却期限が表示されること
5. [ ] 貸出完了メッセージが表示されること
6. [ ] エラー時に警告が表示されること
